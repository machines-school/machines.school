## What is MCP?

MCP is a protocol for connecting AI models to external tools and services. It allows AI models to interact with external systems, such as databases, APIs, and other services, in a standardized way.

### MCP Server

MCP Server is a server that implements the MCP protocol. It is responsible for receiving requests from the client and sending responses back.

```rs
struct Deck {
    cards: Vec<String>,
}

fn main() {
    let suits = vec!["Hearts", "Spades", "Diamonds"];
    let values = vec![ "Ace", "2", "3", "4", "5", "6", "7"];

    let mut cards = vec![];

    for suit in suits {
        for value in values {
            let card = format!("{} of {}", value, suit);
            cards.push(card);
        }
    }

    let deck = Deck { cards: vec![] };

    println!("Here your deck is: {:?}", deck);
}

```
